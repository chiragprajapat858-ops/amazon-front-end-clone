# amazon-front-end-clone
A high-fidelity clone of the Amazon homepage built using HTML5 and CSS3. This project focuses on replicating complex UI components, responsive layouts, and the iconic Amazon aesthetic.

✨ Key Features
Interactive Navbar: Includes a location selector, a robust search bar with category selection, and account/order links.

Responsive Product Grids: A multi-layered shop section featuring a 4-column layout for product categories.

Hover Effects: Precision-styled borders and transitions that mimic the real Amazon user interface.

Comprehensive Footer: Multi-column informational links and a localized language/currency selector.

Hero Section: A visually appealing hero image with an integrated shopping prompt message.

🛠️ Tech Stack
HTML5: For structured and semantic web content.

CSS3: Advanced styling using Flexbox, custom properties, and specific positioning.

FontAwesome: Integrated via CDN for high-quality scalable iconography.
